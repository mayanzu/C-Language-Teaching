循环结构综合应用

掌握 C 语言循环结构的综合应用技巧

循环结构综合应用

循环结构是C语言中最强大的控制结构之一，通过循环可以重复执行代码，实现复杂的算法和数据处理。

新手注意：
  • 循环要有明确的退出条件，避免死循环
  • break跳出整个循环，continue跳过本次循环
  • 嵌套循环要小心控制变量的作用域
重要提示：掌握循环控制是编程的基础，循环可以解决很多实际问题。

while循环进阶

while循环的应用场景

while循环适用于不确定循环次数的情况，特别是需要根据条件动态决定是否继续循环时。

常见应用场景

  • 输入验证：重复提示用户输入，直到输入合法
  • 交互式菜单：程序持续运行直到用户选择退出
  • 数据处理：处理未知数量的数据
  • 游戏循环：游戏持续运行直到满足结束条件
用户输入验证
// while循环输入验证
#include <stdio.h>

int main() {
    int num;
    
    printf("请输入一个1-10之间的数字: ");
    scanf("%d", &num);
    
    // 输入不合法就一直循环
    while (num < 1 || num > 10) {
        printf("输入无效，请重新输入: ");
        scanf("%d", &num);
    }
    
    printf("输入正确: %d\n", num);
    
    return 0;
}
循环设计要点：

• 确保循环条件最终会变为假，避免死循环
• 循环变量要在循环体内被修改
• 复杂条件可以用标志变量控制

练习2：求最大公约数

使用辗转相除法（欧几里得算法），用while循环求两个正整数的最大公约数。

查看参考答案
🔍 欧几里得算法（辗转相除法）数学原理：
  • 核心定理：两个正整数的最大公约数等于其中较小的数和两数相除余数的最大公约数
    • 数学表示：gcd(a, b) = gcd(b, a mod b)，其中 b ≠ 0
    • 终止条件：当 b = 0 时，gcd(a, 0) = a
  • 为什么这个算法是正确的（证明思路）：
    • 设 d 是 a 和 b 的公约数，即 d | a 且 d | b（d整除a和b）
    • 根据带余除法：a = b × q + r，其中 r = a mod b
    • 因为 d | a 且 d | b，所以 d | (a - b×q)，即 d | r
    • 因此 d 也是 b 和 r 的公约数
    • 反过来，b 和 r 的公约数也必然是 a 和 b 的公约数
    • 所以 gcd(a, b) = gcd(b, r) = gcd(b, a mod b)
  • 执行过程示例（求 gcd(48, 18)）： 
    第1步: gcd(48, 18) → 48 ÷ 18 = 2 余 12
      a=48, b=18, r=12 → gcd(18, 12)

第2步: gcd(18, 12) → 18 ÷ 12 = 1 余 6
      a=18, b=12, r=6  → gcd(12, 6)

第3步: gcd(12, 6)  → 12 ÷ 6 = 2 余 0
      a=12, b=6,  r=0  → gcd(6, 0)

第4步: gcd(6, 0) = 6 （b为0，循环结束，a即为最大公约数）

结果: gcd(48, 18) = 6
  • 时间复杂度：
    • 最坏情况：O(log min(a, b))，非常高效
    • 最坏情况发生在连续的斐波那契数，如 gcd(89, 55)
    • 每次迭代至少使数值减半，因此迭代次数是对数级的
#include <stdio.h>

int main() {
    int a, b, remainder;
    
    printf("请输入两个正整数: ");
    scanf("%d %d", &a, &b);
    
    int m = a, n = b;
    
    // 辗转相除法
    while (b != 0) {
        remainder = a % b;
        a = b;
        b = remainder;
    }
    
    printf("%d 和 %d 的最大公约数是 %d\n", m, n, a);
    
    return 0;
}

while循环的执行流程

while循环的执行顺序：先判断条件，再执行循环体。

while 循环执行流程：

1 开始循环
2 判断循环条件
3 条件为真：执行循环体
4 返回步骤2，重新判断
5 条件为假：退出循环
while循环执行流程示例
// 执行流程演示
#include <stdio.h>

int main() {
    int count = 1;
    
    printf("开始while循环\n");
    while (count <= 3) {
        printf("第%d次循环\n", count);
        count++;
    }
    printf("循环结束\n");
    
    return 0;
}

嵌套循环的工作原理

外层循环每执行一次，内层循环就会完整执行一遍。

嵌套循环执行流程：

1 外层循环开始，初始化外层变量
2 判断外层循环条件
3 进入内层循环，初始化内层变量
4 执行内层循环体（完整执行）
5 内层循环结束，返回外层
6 外层循环继续，重复步骤3-5
嵌套循环执行演示
// 演示嵌套循环的执行过程
#include <stdio.h>

int main() {
    int i, j;
    printf(for (i =span> (int i = 1; i <= 2; i++) {
        printf(for (j =span> (int j = 1; j <= 3; j++) {
            printf("  内层第%d次循环\n", j);
        }
        printf("\n");
    }
    
    return 0;
}

break与continue

break语句

break语句用于立即终止当前循环，程序会跳出循环体，继续执行循环后面的语句。

break的工作原理

  • 作用范围：只能跳出最内层的循环（如果有多层嵌套）
  • 执行效果：遇到break立即停止循环，不再执行后续迭代
  • 适用场景：找到目标、遇到错误、满足特定条件需要提前结束
break基本示例
// break跳出循环
#include <stdio.h>

int main() {
    int i;
    printf(for (i =n> (int i = 1; i <= 10; i++) {
        if (i == 5) {
            printf("\n遇到5，使用break跳出！\n");
            break;  // 遇到5时跳出循环
        }
        printf("%d ", i);
    }
    printf("循环结束\n");
    
    return 0;
}
// 输出：1 2 3 4

continue语句

continue语句用于跳过本次循环中continue后面的语句，直接开始下一次循环迭代。

continue的工作原理

  • 作用范围：只影响它所在的那一层循环
  • 执行效果：跳过本次循环剩余的代码，但循环不会结束
  • 适用场景：过滤数据、跳过特定条件的情况、避免深层嵌套的if语句
continue基本示例
// continue跳过本次循环
#include <stdio.h>

int main() {
    int i;
    printf(for (i =n> (int i = 1; i <= 10; i++) {
        if (i % 3 == 0) {
            continue;  // 跳过3的倍数
        }
        printf("%d ", i);
    }
    printf("\n");
    
    return 0;
}
// 输出：1 2 4 5 7 8 10（跳过了3, 6, 9）

continue与条件判断的配合

continue常用于简化代码逻辑，避免过多的if-else嵌套。

break与continue的区别

关键区别

语句 作用 循环是否继续 适用场景
break 完全终止循环 否，跳出整个循环 找到目标后停止搜索，错误处理等
continue 跳过本次循环剩余代码 是，继续下一次迭代 跳过某些不符合条件的元素
break vs continue对比
// break vs continue对比
#include <stdio.h>

int main() {
    int i;
    // 使用break
    printf(for (i =n> (int i = 1; i <= 5; i++) {
        if (i == 3) {
            break;  // 遇到3就停止
        }
        printf("%d ", i);
    }
    printf("\n");  // 输出：1 2
    
    // 使用continue
    printf(for (i =n> (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue;  // 跳过3，继续循环
        }
        printf("%d ", i);
    }
    printf("\n");  // 输出：1 2 4 5
    
    return 0;
}
