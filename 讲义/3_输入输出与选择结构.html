输入输出与选择结构

掌握程序与用户交互和条件判断的方法

输入输出与选择结构基础

C语言程序需要与用户进行交互，这主要通过输入输出函数实现。同时，程序需要根据不同条件执行不同操作，这就需要选择结构（条件判断语句）。

生活类比——餐厅点餐：

把程序的输入输出想象成餐厅点餐：

  • 输入（scanf）：顾客告诉服务员想要什么菜（用户向程序提供数据）
  • 处理（选择结构）：厨师根据菜单决定怎么做（程序根据条件做判断）
  • 输出（printf）：服务员把做好的菜端上来（程序把结果显示给用户）

没有输入，程序就像没有顾客的餐厅；没有输出，就像做好了菜却不端上来。

新手注意：
  • 输入输出是程序与用户沟通的桥梁
  • 条件判断让程序能够做出选择
  • 格式控制是 C 语言输入输出的重点

什么是输入输出？

输入（Input）是指程序从外部获取数据的过程。最常见的输入方式是用户通过键盘输入数据，程序用 scanf 函数读取这些数据。

输出（Output）是指程序向外部展示数据的过程。最常见的输出方式是程序把结果显示在屏幕上，用 printf 函数实现。

什么是选择结构？

选择结构（也叫条件判断）让程序能够根据不同的情况执行不同的代码。就像生活中的"如果...那么..."：

  • 如果下雨，那么带伞
  • 如果成绩及格，那么显示"通过"，否则显示"不通过"
  • 如果是周末，那么可以睡懒觉

选择结构让程序有了"思考"的能力，不再只是简单地从头执行到尾。

学习目标与节奏

按照下面的节奏来学习，确保每一步都扎实：

  1. 第1步：理解格式化输出 —— 掌握 printf 的基本用法和格式说明符
  2. 第2步：学习数据输入 —— 理解 scanf 的工作原理和地址传递
  3. 第3步：条件判断基础 —— 学会用 if 语句做简单判断
  4. 第4步：多分支选择 —— 掌握 switch 语句的使用方法
  5. 第5步：综合练习 —— 结合输入输出和选择结构写完整程序

printf函数详解

printf基本用法

printf 是 C 语言中最常用的输出函数，用于向屏幕输出格式化的文本。它是 "print formatted" 的缩写，定义在 stdio.h 头文件中。

生活类比——填空题：

printf 就像做填空题：

  • 格式字符串：题目中的句子，有若干空格需要填写
  • 格式说明符：每个空格的类型标记（填数字、填文字等）
  • 参数列表：你要填入空格的具体内容

例如：printf("我叫%s，今年%d岁", "小明", 18);

就像题目："我叫____，今年____岁"，你填入"小明"和"18"。

基本语法

printf("格式字符串", 参数列表);

  • 格式字符串：包含要输出的文本和格式说明符（如 %d、%f 等）
  • 参数列表：要输出的变量，按照格式说明符的顺序提供

printf的工作原理

printf 函数的工作过程可以理解为：

  1. 扫描格式字符串：从头到尾读取格式字符串
  2. 遇到普通字符：直接输出到屏幕
  3. 遇到格式说明符：从参数列表中取出对应的值，按照指定格式输出
  4. 重复以上步骤：直到格式字符串结束
printf基本示例
// printf基本用法示例
#include <stdio.h>

int main() {
    // 简单文本输出 - 不需要格式说明符
    printf("Hello, World!\n");

    // 格式化输出整数
    int age = 20;
    printf("年龄: %d\n", age);  // %d是整数的格式说明符

    // 格式化输出浮点数
    float height = 1.75;
    printf("身高: %.2f 米\n", height);  // %.2f保留2位小数

    // 格式化输出字符
    char grade = 'A';
    printf("等级: %c\n", grade);  // %c是字符的格式说明符

    // 一次输出多个变量
    int score = 95;
    printf("学生年龄%d岁，成绩%d分，等级%c\n", age, score, grade);

    return 0;
}
重要提示：

printf 的格式说明符数量必须与后面的参数数量一致，类型也必须匹配。\n 表示换行，如果不加 \n，下一次输出会接在同一行。

常用格式说明符

格式说明符决定了数据以什么形式输出，是 printf 最核心的知识点。

格式符 含义 示例 输出结果
%d 十进制整数 printf("%d", 42); 42
%f 浮点数（默认6位小数） printf("%f", 3.14); 3.140000
%.2f 保留2位小数 printf("%.2f", 3.14159); 3.14
%c 单个字符 printf("%c", 'A'); A
%s 字符串 printf("%s", "hello"); hello
%%
%o 八进制整数 printf("%o", 10); 12
%x 十六进制整数（小写） printf("%x", 255); ff
%u 无符号十进制整数 printf("%u", -1); 4294967295
%5d 宽度5，右对齐 printf("%5d", 42); ___42（3个空格+42）
%-5d 宽度5，左对齐 printf("%-5d", 42); 42___（42+3个空格）
%05d 宽度5，补零 printf("%05d", 42); 00042
输出百分号本身 printf("100%%"); 100%
格式说明符实战示例
// 各种格式说明符的使用示例
#include <stdio.h>

int main() {
    int age = 18;
    float score = 95.5;
    char grade = 'A';
    char name[20] = "小明";

    // 基本格式输出
    printf("姓名: %s\n", name);
    printf("年龄: %d 岁\n", age);
    printf("成绩: %.1f 分\n", score);
    printf("等级: %c 级\n", grade);

    // 多个变量一起输出
    printf("%s今年%d岁，成绩%.1f分，等级%c\n", name, age, score, grade);

    // 输出百分号
    printf("完成度: 100%%\n");

    return 0;
}

宽度对齐与精度控制

printf 支持多种格式控制，让输出更整齐美观。

宽度与对齐

通过数字指定输出宽度，- 表示左对齐，0 表示前导零填充。

宽度和对齐控制
// 宽度和对齐控制示例
#include <stdio.h>

int main() {
    int num1 = 42, num2 = 123;

    // 宽度控制 - 右对齐（默认）
    printf("右对齐: '%5d'\n", num1);  // '   42'
    printf("右对齐: '%5d'\n", num2);  // '  123'

    // 左对齐
    printf("左对齐: '%-5d'\n", num1);  // '42   '
    printf("左对齐: '%-5d'\n", num2);  // '123  '

    // 前导零
    printf("前导零: '%05d'\n", num1);  // '00042'

    return 0;
}
浮点数精度控制
// 浮点数精度控制示例
#include <stdio.h>

int main() {
    double pi = 3.141592653589793;
    double money = 123.456789;

    // 默认精度（6位小数）
    printf("默认: %f\n", pi);  // 3.141593

    // 指定小数位数
    printf("2位小数: %.2f\n", pi);  // 3.14
    printf("4位小数: %.4f\n", pi);  // 3.1416

    // 货币格式
    printf("金额: %.2f 元\n", money); // 123.46

    return 0;
}

知识点回顾：

scanf函数详解

scanf基本用法

scanf 是 C 语言中最常用的输入函数，用于从键盘读取用户输入。它是 "scan formatted" 的缩写，定义在 stdio.h 头文件中。

基本语法

scanf("格式字符串", &变量1, &变量2, ...);

  • 格式字符串：指定输入数据的类型和格式
  • &变量：取地址运算符 & 后跟变量名，表示变量的内存地址
scanf基本示例
// scanf基本用法示例
#include <stdio.h>

int main() {
    int age;
    float height;
    char grade;
    char name[50];

    printf("请输入年龄: ");
    scanf("%d", &age);

    printf("请输入身高: ");
    scanf("%f", &height);

    printf("请输入等级: ");
    scanf(" %c", &grade);  // 注意%c前面的空格

    printf("请输入姓名: ");
    scanf("%s", name);  // 字符串不需要&

    printf("\n输入信息:\n");
    printf("年龄: %d\n", age);
    printf("身高: %.2f\n", height);
    printf("等级: %c\n", grade);
    printf("姓名: %s\n", name);

    return 0;
}

地址运算符&

& 是取地址运算符，scanf 需要知道变量在内存中的地址才能把数据存进去。

为什么需要 & ？

scanf 要"存"数据，需要知道"存"在哪里，所以要用 & 获取地址。printf 只是"读"数据，不需要地址。

重要提醒：
  • scanf 读取整数、浮点数、字符时都需要 &
  • 读取字符串时不需要 &（因为数组名本身就是地址）
  • 忘记 & 是最常见的错误，会导致程序崩溃
正确与错误的对比
// 正确与错误的对比
#include <stdio.h>

int main() {
    int num;

    // 错误：缺少&
    // scanf("%d", num);  // 编译错误或程序崩溃！

    // 正确：使用&
    scanf("%d", &num);
    printf("输入的数字是: %d\n", num);

    // 字符串数组名本身就是地址，不需要&
    char str[100];
    scanf("%s", str);  // 正确

    return 0;
}
记忆口诀：

scanf 读数据要地址，& 符号不能少。字符串数组不用 &，数组名就是地址。

scanf常用格式符

scanf 的格式符与 printf 类似，但也有一些不同之处。

格式符 用途 示例
%d 读取整数 scanf("%d", &num);
%f 读取 float 浮点数 scanf("%f", &height);
%lf 读取 double 浮点数 scanf("%lf", &money);
%c 读取单个字符 scanf(" %c", &grade);
%s 读取字符串（遇空格停止） scanf("%s", name);

输入缓冲区问题

scanf 读取数据时，输入的内容会先存入"缓冲区"，scanf 从缓冲区中读取。如果缓冲区中有残留数据，可能导致读取异常。

常见问题：读取整数后再读字符

scanf 读取整数后，输入缓冲区会残留换行符 \n，导致下一个 %c 直接读到换行符。

解决方法：在 %c 前加空格：scanf(" %c", &ch);

常见问题：scanf 不能读取带空格的字符串

scanf 遇到空格就会停止读取。如果要读取一整行，通常使用 fgets()。gets() 已被废弃，不建议使用。

scanf 的返回值（二级考点）：

scanf 返回成功读取的参数个数。读取失败或遇到文件末尾时返回 EOF（值为 -1）。

int a, b;
int ret = scanf("%d%d", &a, &b);
// 如果输入 "10 20"，ret = 2（成功读取2个）
// 如果输入 "10 abc"，ret = 1（只成功读取1个）

// 二级常见用法：循环读取直到文件末尾
while (scanf("%d", &a) != EOF) {
    printf("%d\n", a);
}
// 这种写法常出现在程序填空题中
输入缓冲区问题示例
// 输入缓冲区问题示例
#include <stdio.h>

int main() {
    int num;
    char ch;

    printf("请输入一个整数: ");
    scanf("%d", &num);

    // 错误：缓冲区中残留\n，ch会读到\n
    // scanf("%c", &ch);

    // 正确：%c前加空格，跳过空白字符
    printf("请输入一个字符: ");
    scanf(" %c", &ch);  // 注意%c前面的空格

    printf("整数: %d, 字符: %c\n", num, ch);

    return 0;
}
安全注意事项：
  • 读取字符串时，应限制输入长度防止缓冲区溢出：scanf("%9s", str);
  • 不要使用 gets() 函数，它无法限制输入长度，已被 C11 标准移除
  • 推荐使用 fgets() 读取一整行：fgets(str, sizeof(str), stdin);

if语句

单分支if

if 语句用于根据条件执行不同的代码块，是程序实现"判断和选择"的基础。单分支 if 只有一个判断条件，满足条件时执行代码块。

生活类比——门禁系统：

单分支 if 就像小区的门禁：

  • 条件：你是小区的业主吗？
  • 满足条件：门打开，你可以进入
  • 不满足条件：门不开，你只能站在外面（程序继续往下执行）

不管门开不开，小区里的其他活动（程序后续代码）都会正常进行。

语法格式

if (条件) { 语句块; }

  • 条件：一个能产生真假值的表达式（关系表达式或逻辑表达式）
  • 语句块：条件为真时要执行的代码，用大括号包围
单分支if示例
// 单分支if：只有"满足条件执行"
#include <stdio.h>

int main() {
    int age;

    printf("请输入年龄: ");
    scanf("%d", &age);

    if (age >= 18) {
        printf("您已成年，可以观看此内容。\n");
    }

    printf("程序结束。\n");

    return 0;
}
执行流程：
  1. 程序执行到 if 语句
  2. 计算括号中条件的值
  3. 如果条件为真（非0），执行大括号内的语句块
  4. 如果条件为假（0），跳过大括号内的语句块
  5. 继续执行 if 语句后面的代码

双分支if-else

双分支 if-else 有两个分支：条件满足执行 if 块，不满足执行 else 块。

生活类比——红绿灯：

双分支 if-else 就像过马路看红绿灯：

  • 条件：灯是绿灯吗？
  • 满足条件（绿灯）：过马路
  • 不满足条件（红灯）：停下来等待

两种情况必选其一，不可能同时发生。

语法格式

if (条件) { 语句块1; } else { 语句块2; }

  • 条件为真：执行语句块1，跳过语句块2
  • 条件为假：跳过语句块1，执行语句块2
  • 重要特点：两个语句块只会执行其中一个，不会都执行，也不会都不执行
双分支if-else示例
// 双分支if-else：满足和不满足各有一个分支
#include <stdio.h>

int main() {
    int num;

    printf("请输入一个整数: ");
    scanf("%d", &num);

    if (num % 2 == 0) {
        printf("%d是偶数\n", num);
    } else {
        printf("%d是奇数\n", num);
    }

    return 0;
}
常见应用场景：
  • 判断奇偶数：if (num % 2 == 0) ... else ...
  • 判断及格/不及格：if (score >= 60) ... else ...
  • 判断正负数：if (num > 0) ... else ...
  • 判断闰年：if ((year%4==0 && year%100!=0) || year%400==0) ... else ...

多分支if-else if

有多个条件判断时，使用 if-else if 结构，从上到下依次检查，执行第一个满足条件的分支。

多分支if-else if示例
// 多分支if-else if：多种情况分别处理
#include <stdio.h>

int main() {
    int score;

    printf("请输入成绩: ");
    scanf("%d", &score);

    // 从上到下依次检查条件
    if (score >= 90) {
        printf("优秀！继续保持！\n");
    } else if (score >= 80) {
        printf("良好！再接再厉！\n");
    } else if (score >= 60) {
        printf("及格了，要继续努力！\n");
    } else {
        printf("不及格，需要加油！\n");
    }

    return 0;
}
执行顺序：

if-else if 结构从上到下依次检查条件，一旦找到满足的条件就执行对应代码块，然后跳过所有剩余分支。即使后面还有满足的条件也不会再执行。

条件表达式详解

if 语句的条件可以是任何能产生真（非0）或假（0）的表达式。

  • 比较运算：a > b, a == b, a != b
  • 逻辑运算：a > 0 && b < 10, a == 0 || b == 0（逻辑运算符详见第2章）
  • 直接判断：if (num) 等价于 if (num != 0)
知识点回顾：

逻辑运算符 &&（逻辑与）、||（逻辑或）、!（逻辑非）及其真值表已在第2章详细介绍，此处不再重复。在条件表达式中最常用的是 && 和 ||，用于组合多个条件。

短路特性：

&&：左边为假时，右边不执行。||：左边为真时，右边不执行。利用这个特性可以避免一些错误，例如：if (x != 0 && 10 / x > 1) 不会产生除零错误。

switch语句

switch基本语法

switch 语句适用于多分支选择，特别是当分支较多且基于同一个表达式的不同值时。它比 if-else if 更清晰、更高效。

生活类比——自动售货机：

switch 就像自动售货机的按钮选择：

  • 表达式：你按下的按钮编号
  • case：每个按钮对应的饮料
  • break：拿到饮料后离开
  • default：按下无效按钮，提示"请重新选择"

你按哪个按钮，就得到对应的饮料。每个按钮对应一种结果，清晰明了。

语法格式

switch (表达式) { case 值1: 语句1; break; case 值2: 语句2; break; ... default: 默认语句; }

  • switch 表达式：必须是整数类型（int、char 等），不能是 float 或 double
  • case 标签：必须是常量表达式，不能是变量
  • break：跳出 switch 结构，防止"贯穿"执行
  • default：所有 case 都不匹配时执行，类似于 if-else if 末尾的 else

switch的执行流程

  1. 计算 switch 后面括号中表达式的值
  2. 从上到下依次与每个 case 的值比较
  3. 找到匹配的 case，执行该 case 后面的语句
  4. 遇到 break，跳出整个 switch 结构
  5. 如果没有匹配的 case，执行 default（如果有的话）
switch基本用法
// switch语句基本用法
#include <stdio.h>

int main() {
    int day;

    printf("请输入星期几(1-7): ");
    scanf("%d", &day);

    switch(day) {
        case 1:
            printf("星期一\n");
            break;
        case 2:
            printf("星期二\n");
            break;
        case 3:
            printf("星期三\n");
            break;
        case 4:
            printf("星期四\n");
            break;
        case 5:
            printf("星期五\n");
            break;
        case 6:
            printf("星期六\n");
            break;
        case 7:
            printf("星期日\n");
            break;
        default:
            printf("输入错误\n");
    }

    return 0;
}

break的作用

break 语句用于跳出 switch 结构。如果 case 分支末尾没有 break，程序会"贯穿"到下一个 case 继续执行，这通常是 bug 的来源。

缺少break的贯穿现象
// 缺少break的贯穿现象
#include <stdio.h>

int main() {
    int num = 2;

    switch(num) {
        case 1:
            printf("一\n");
            // 缺少break，会贯穿到case 2
        case 2:
            printf("二\n");
            // 缺少break，会贯穿到case 3
        case 3:
            printf("三\n");
            break;
    }
    // 输出：二  三（从case 2开始贯穿执行）

    return 0;
}
注意：

每个 case 分支末尾都应该有 break，除非你明确需要利用贯穿特性。忘记 break 是 switch 语句最常见的错误之一。

case合并与default子句

多个 case 可以执行相同的代码，将它们合并在一起。default 子句是可选的，用于处理所有 case 都不匹配的情况。

case合并示例 - 判断季节

switch与if-else if对比

switch 和 if-else if 都可以实现多分支选择，但适用场景不同。

适合用 switch
  • 基于整数值的多分支判断
  • 分支较多（4个以上）
  • 每个条件都是相等比较
  • 代码更清晰，执行效率更高
适合用 if-else if
  • 基于范围的判断（如 score >= 90）
  • 分支较少（2-3个）
  • 需要复杂条件（逻辑运算）
  • 条件涉及浮点数比较
switch vs if-else if对比
// 使用switch的情况 - 基于整数值
int choice = 2;
switch(choice) {
    case 1: printf("选项1\n"); break;
    case 2: printf("选项2\n"); break;
    case 3: printf("选项3\n"); break;
}

// 使用if-else if的情况 - 基于范围
int score = 85;
if (score >= 90) {
    printf("优秀\n");
} else if (score >= 80) {
    printf("良好\n");
} else if (score >= 60) {
    printf("及格\n");
}

选择结构嵌套

悬空 else 问题（二级经典题）：

当有多个 if 但只有一个 else 时，else 与最近的、尚未匹配的 if 配对：

if (a > 0)
    if (b > 0)
        printf("a>0且b>0\n");
    else
        printf("???\n");  // 这个else跟哪个if配对？
// 答案：跟 if (b > 0) 配对（最近的if），不是跟 if (a > 0)
// 虽然缩进暗示else属于外层if，但C语言不看缩进！

// 正确写法：用大括号明确
if (a > 0) {
    if (b > 0)
        printf("a>0且b>0\n");
} else {
    printf("a<=0\n");  // 现在else明确属于外层if
}

嵌套if

if 语句可以嵌套使用，实现更复杂的逻辑判断。嵌套时要注意代码的缩进和大括号的配对。

嵌套if语句示例
// 嵌套if语句示例 - 找三个数中的最大值
#include <stdio.h>

int main() {
    int a = 15, b = 20, c = 10;
    int max;

    if (a > b) {
        if (a > c) {
            max = a;
        } else {
            max = c;
        }
    } else {
        if (b > c) {
            max = b;
        } else {
            max = c;
        }
    }

    printf("三个数中的最大值是: %d\n", max);

    return 0;
}

嵌套注意事项

嵌套使用注意：
  • else 的配对原则：else 总是与最近的、未配对的 if 配对，与缩进无关
  • 大括号很重要：加上大括号可以明确代码块的范围，避免歧义
  • 避免过深嵌套：嵌套超过3层时，考虑用逻辑运算符简化或拆分函数
  • 保持缩进一致：良好的缩进让代码更易读
else配对问题示例
// else的配对问题
#include <stdio.h>

int main() {
    int x = 5, y = 3;

    // 错误的理解：else与外层if配对？
    // 实际上：else与最近的if（内层）配对
    if (x > 0)
        if (y > 10)
            printf("A\n");
    else  // 这个else与内层if配对，不是外层！
        printf("B\n");

    // 正确写法：用大括号明确配对关系
    if (x > 0) {
        if (y > 10) {
            printf("A\n");
        }
    } else {  // 这个else与外层if配对
        printf("B\n");
    }

    return 0;
}

常见错误

最容易犯的错误

初学者在学习输入输出和选择结构时，容易犯以下错误，务必注意避免。

错误1：忘记scanf的&
  • 读取整数、浮点数、字符时忘记加 &
  • 会导致程序崩溃或读取到随机值
  • 只有字符串（数组名）不需要 &
错误2：= 和 == 混淆
  • = 是赋值，== 是比较
  • if (x = 5) 是赋值，永远为真
  • 应写为 if (x == 5)
错误3：缺少大括号
  • if/else 只管一条语句，多条语句必须加 {}
  • 即使只有一条语句，也建议加 {}
  • 良好的代码习惯能减少 bug

错误代码对比

以下是常见错误的错误写法和正确写法对比：

错误1：忘记scanf的&
// 错误
int num;
scanf("%d", num);  // 缺少&，程序崩溃！

// 正确
scanf("%d", &num);
错误2：= 和 == 混淆
// 错误
if (x = 5) {  // 赋值，x变为5，条件永远为真
    printf("x等于5");
}

// 正确
if (x == 5) {  // 比较，判断x是否等于5
    printf("x等于5");
}
错误3：缺少大括号
// 错误：else只管一条语句
if (score >= 60)
    printf("及格\n");
    printf("恭喜！\n");  // 这行总会执行，不在else控制范围内
else  // 编译错误：else找不到对应的if
    printf("不及格\n");

// 正确：用大括号明确代码块
if (score >= 60) {
    printf("及格\n");
    printf("恭喜！\n");
} else {
    printf("不及格\n");
}
错误4：错误的范围判断
// 错误：数学中正确，C语言中错误
if (0 < x < 100) {  // 先算 0
    printf("在范围内");
}

// 正确：使用逻辑与运算符
if (x > 0 && x < 100) {
    printf("在范围内");
}

字符输入输出

getchar与putchar

除了 printf 和 scanf，C语言还提供了专门用于单个字符输入输出的函数。

getchar和putchar基本使用
#include <stdio.h>

int main() {
    char ch;

    // getchar(): 从键盘读取一个字符
    printf("请输入一个字符: ");
    ch = getchar();

    // putchar(): 向屏幕输出一个字符
    printf("你输入的字符是: ");
    putchar(ch);
    putchar('\n');  // 输出换行

    return 0;
}
与scanf的区别：
  • getchar() 一次只读一个字符，不需要格式字符串
  • scanf("%c", &ch) 也能读字符，但 getchar() 更简洁
  • putchar(ch) 比 printf("%c", ch) 更高效，因为不需要解析格式字符串

清除输入缓冲区

使用 scanf 读取数字后，换行符会留在缓冲区中，影响后续的字符读取。可以用 getchar() 来清除：

清除输入缓冲区残留
#include <stdio.h>

int main() {
    int age;
    char grade;

    printf("请输入年龄: ");
    scanf("%d", &age);

    // 清除缓冲区中的换行符
    while (getchar() != '\n')
        ;  // 空循环体，读完所有残留字符

    printf("请输入等级(A/B/C): ");
    scanf("%c", &grade);

    printf("年龄: %d, 等级: %c\n", age, grade);

    return 0;
}
边学边练 - 字符输入输出

练习1： 写一个程序，使用getchar读取一个字符，如果它是大写字母就转成小写输出，如果是小写字母就转成大写输出。

参考答案 
#include <stdio.h>

int main() {
    char ch = getchar();

    if (ch >= 'A' && ch <= 'Z') {
        putchar(ch + 32);  // 大写转小写
    } else if (ch >= 'a' && ch <= 'z') {
        putchar(ch - 32);  // 小写转大写
    } else {
        putchar(ch);  // 非字母原样输出
    }

    return 0;
}

条件运算符与选择结构

用条件运算符简化选择

在第2章我们学过三元条件运算符 ? :，它在选择结构中有很多实用场景。

求最大值/最小值

用条件运算符求最大值
int a = 10, b = 20;
int max = (a > b) ? a : b;  // max = 20
int min = (a < b) ? a : b;  // min = 10
printf("最大值: %d, 最小值: %d\n", max, min);

判断奇偶

条件运算符判断奇偶
int num = 7;
printf("%d是%s数\n", num, num % 2 == 0 ? "偶" : "奇");
可读性建议：

条件运算符适合简单的二选一场景。如果逻辑复杂或嵌套多层，建议使用 if-else，可读性更重要。

选择结构设计思路

面对一个需要条件判断的问题，应该遵循以下思路：

步骤一：明确所有可能的情况

先列出所有可能的分支，不要遗漏。

步骤二：选择合适的语句

场景 推荐语句 原因
两个互斥条件if-else结构清晰
多个连续区间判断if-else if灵活表达区间
离散值多分支switch结构清晰，效率高
简单二选一赋值条件运算符代码简洁

步骤三：注意边界条件

边界条件示例
// 判断成绩等级 - 注意边界用 >= 还是 >
int score = 90;

// 正确：使用 >= 包含边界值
if (score >= 90) {
    printf("优秀\n");
} else if (score >= 80) {
    printf("良好\n");
} else if (score >= 60) {
    printf("及格\n");
} else {
    printf("不及格\n");
}
