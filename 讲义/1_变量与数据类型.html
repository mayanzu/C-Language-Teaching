理解C语言如何存储和处理不同类型的数据，这是编程的基础

变量与数据类型基础

在上一章中，我们学习了计算机的基础知识和第一个C语言程序。现在，我们要深入了解C语言如何存储和处理数据——这就是变量和数据类型。

新手注意：
  • 变量就像一个带标签的盒子，用来存放数据
  • 数据类型决定了盒子的大小和能存放什么样的数据
  • 使用变量前必须先声明（告诉编译器盒子的类型和名字）
重要提示：

掌握变量和数据类型是写好C程序的前提，就像认识字才能写文章一样。本章将介绍C语言的基本数据类型和变量的使用方法。

变量基础

什么是变量？

变量是程序中用来存储数据的容器。你可以把它想象成一个带标签的盒子：

  • 变量名就是盒子上的标签，用来找到这个盒子
  • 变量的值就是盒子里装的东西
  • 变量的类型决定了盒子的大小和能装什么样的东西
生活类比：

想象你要存放不同物品：

  • 整数值（如年龄25）→ 用小盒子（int）
  • 小数值（如身高1.75）→ 用精度盒子（float/double）
  • 单个字符（如字母'A'）→ 用迷你盒子（char）

变量的声明与初始化

使用变量前，必须先声明它——告诉编译器变量的类型和名字。

变量声明与初始化
#include <stdio.h>

int main() {
    // 声明变量（只创建盒子，不放入内容）
    int age;

    // 声明并初始化（创建盒子并放入内容）
    int score = 95;

    // 同时声明多个同类型变量
    int a = 1, b = 2, c = 3;

    // 先声明，后赋值
    float height;
    height = 1.75;

    printf("年龄: %d\n", age);    // 注意：未初始化的变量值不确定！
    printf("成绩: %d\n", score);
    printf("身高: %.2f\n", height);

    return 0;
}
重要警告：

未初始化的局部变量包含垃圾值（不确定的随机值），使用前一定要初始化！

变量的命名规则

C语言对变量名有严格的规则：

规则 说明 正确示例 错误示例
可用字符 字母、数字、下划线 age、num_1 my-age、2nd
不能数字开头 第一个字符必须是字母或下划线 _count、num2 2num
不能是关键字 如int、float、return等 myInt int
区分大小写 Age和age是不同的变量 Age、age —
命名建议：
  • 变量名要有意义，见名知意（如 student_age 而非 a）
  • 使用下划线风格（snake_case）或小驼峰风格（camelCase）
  • 避免使用单字母变量名（循环变量 i、j、k 除外）

整型（Integer）

基本整型 int

int 是C语言中最常用的整数类型，用于存储没有小数部分的数值。

int 类型示例
#include <stdio.h>

int main() {
    int age = 20;          // 年龄
    int temperature = -5;   // 温度（可以是负数）
    int count = 0;           // 计数器

    printf("年龄: %d\n", age);
    printf("温度: %d\n", temperature);
    printf("计数: %d\n", count);

    return 0;
}

整型的分类

C语言提供了多种整型，它们的主要区别是占用的内存大小和能表示的范围：

类型 占用字节 范围 用途
short 2字节 -32768 ~ 32767 节省空间的小整数
int 4字节 -2147483648 ~ 2147483647 一般整数（最常用）
long 4/8字节 更大范围 大整数
long long 8字节 极大范围 非常大的整数
unsigned int 4字节 0 ~ 4294967295 非负整数
新手注意：
  • 日常编程中使用 int 就足够了
  • unsigned 表示无符号，只能存非负数
  • 可以用 sizeof 运算符查看类型占用的字节数
使用sizeof查看类型大小
#include <stdio.h>

int main() {
    printf("short: %d 字节\n", sizeof(short));
    printf("int: %d 字节\n", sizeof(int));
    printf("long: %d 字节\n", sizeof(long));
    printf("long long: %d 字节\n", sizeof(long long));

    return 0;
}

整数的表示方式

C语言中整数可以用不同的进制表示：

进制 前缀 示例 十进制值
十进制 无 42 42
八进制 0 052 42
十六进制 0x或0X 0x2A 42
不同进制表示
#include <stdio.h>

int main() {
    int dec = 42;      // 十进制
    int oct = 052;      // 八进制（前缀0）
    int hex = 0x2A;     // 十六进制（前缀0x）

    printf("十进制: %d\n", dec);  // 42
    printf("八进制: %d\n", oct);  // 42
    printf("十六进制: %d\n", hex); // 42

    printf("八进制显示: %o\n", dec);   // 52
    printf("十六进制显示: %x\n", dec);  // 2a

    return 0;
}

浮点型（Float / Double）

什么是浮点型？

浮点型用于存储带小数部分的数值。C语言提供了两种浮点类型：

类型 占用字节 有效数字 范围 用途
float 4字节 6-7位 ±3.4×10³⁸ 单精度，一般小数
double 8字节 15-16位 ±1.7×10³⁰⁸ 双精度，更精确的小数

浮点型示例

浮点型使用示例
#include <stdio.h>

int main() {
    float height = 1.75f;       // 注意：float类型加f后缀
    double pi = 3.141592653589; // double类型更精确
    float temperature = 36.5f;

    printf("身高: %.2f 米\n", height);      // 保留2位小数
    printf("圆周率: %.10f\n", pi);         // 保留10位小数
    printf("体温: %.1f 度\n", temperature); // 保留1位小数

    // 科学计数法
    double distance = 1.5e8;  // 1.5 × 10⁸ = 150000000
    printf("距离: %e\n", distance);  // 科学计数法显示

    return 0;
}
浮点数精度问题：
  • 浮点数在计算机中是近似存储的，不是精确值
  • 0.1 + 0.2 的结果可能不是精确的 0.3
  • 需要高精度时使用 double 而非 float
  • 不要用 == 直接比较两个浮点数是否相等

字符型（char）

什么是字符型？

char 类型用于存储单个字符。在C语言中，字符本质上是一个小整数，因为计算机用数字来表示字符（ASCII编码）。

字符型基本使用
#include <stdio.h>

int main() {
    char grade = 'A';       // 字符用单引号包围
    char digit = '5';       // 字符'5'和数字5不同！
    char symbol = '+';      // 符号也是字符

    printf("等级: %c\n", grade);    // 以字符形式输出
    printf("ASCII值: %d\n", grade);  // 以整数形式输出ASCII码

    return 0;
}

ASCII码表（常用部分）

字符 ASCII值 字符 ASCII值
'0' - '9' 48 - 57 'A' - 'Z' 65 - 90
'a' - 'z' 97 - 122 ' '（空格） 32
'\n'（换行） 10 '\0'（空字符） 0
字符运算：

因为字符本质是整数，所以可以对字符做算术运算：

  • 大写转小写：小写 = 大写 + 32（如 'A' + 32 = 'a'）
  • 字符转数字：数字 = 字符 - '0'（如 '5' - '0' = 5）

类型转换

自动类型转换

当不同类型的数据参与运算时，C语言会自动进行类型转换，遵循"小类型→大类型"的原则：

自动类型转换
#include <stdio.h>

int main() {
    int a = 10;
    float b = 3.0f;

    // int和float运算，int自动转为float
    float result = a / b;  // 10.0 / 3.0 = 3.333...
    printf("10 / 3.0 = %.2f\n", result);

    // 但两个int相除，结果还是int
    int int_result = a / 3;  // 10 / 3 = 3（小数部分被丢弃）
    printf("10 / 3 = %d\n", int_result);

    // char参与运算时，自动转为int
    char ch = 'A';
    int ch_value = ch + 1;  // 65 + 1 = 66，即'B'
    printf("'A' + 1 = %c\n", ch_value);

    return 0;
}

强制类型转换

当你需要明确地将一种类型转换为另一种类型时，使用强制类型转换（类型转换运算符）：

强制类型转换
#include <stdio.h>

int main() {
    int a = 10, b = 3;

    // 不转换：整数除法
    printf("整数除法: %d\n", a / b);  // 输出: 3

    // 强制转换：浮点除法
    printf("浮点除法: %.2f\n", (float)a / b);  // 输出: 3.33

    // 也可以转换另一个操作数
    printf("浮点除法: %.2f\n", a / (float)b);  // 输出: 3.33

    // double转int：小数部分被截断
    double pi = 3.14159;
    int int_pi = (int)pi;
    printf("pi = %f, (int)pi = %d\n", pi, int_pi);  // 3.141590, 3

    return 0;
}
类型转换规则：
  • 转换方向：char → short → int → long → float → double
  • 从低精度到高精度：安全，不会丢失数据
  • 从高精度到低精度：可能丢失数据（如 double → int 会截断小数）

常量

什么是常量？

常量是程序运行过程中值不能改变的数据。与变量不同，常量一旦定义就不能修改。

常量的定义方式

1. const 关键字
  • 声明时加 const 修饰
  • 有明确的数据类型
  • 可以被调试器检查
  • 推荐使用这种方式
2. 宏定义 #define
  • 编译前文本替换
  • 没有类型检查
  • 不占内存空间
  • 常用于全局常量
常量定义示例
#include <stdio.h>

// 方式1：宏定义（习惯用大写字母）
#define PI 3.14159
#define MAX_SIZE 100

int main() {
    // 方式2：const关键字
    const double E = 2.71828;
    const int DAYS_PER_WEEK = 7;

    // 使用常量
    double area = PI * 5.0 * 5.0;
    printf("圆面积: %.2f\n", area);
    printf("自然常数: %f\n", E);
    printf("每周天数: %d\n", DAYS_PER_WEEK);

    // DAYS_PER_WEEK = 8;  // 错误！常量不能修改

    return 0;
}

字面量常量

程序中直接写出的值也叫字面量：

类型 示例 说明
整型常量 42、-5、0xFF 默认是int类型
浮点常量 3.14、1.5e8 默认是double类型
字符常量 'A'、'\n' 用单引号包围
字符串常量 "Hello" 用双引号包围

练习3： 编写程序，输入一个华氏温度，输出对应的摄氏温度。公式：c = (f - 32) * 5 / 9。注意：要使用浮点除法得到正确结果。

参考答案
温度转换
#include <stdio.h>

int main() {
    double fahrenheit, celsius;

    printf("请输入华氏温度: ");
    scanf("%lf", &fahrenheit);

    // 注意：5.0/9.0 确保浮点除法
    celsius = (fahrenheit - 32) * 5.0 / 9.0;

    printf("摄氏温度: %.1f\n", celsius);

    return 0;
}

练习4： 使用 sizeof 运算符查看你的电脑上各数据类型占用的字节数。

参考答案
查看类型大小
#include <stdio.h>

int main() {
    printf("char:   %d 字节\n", sizeof(char));
    printf("short:  %d 字节\n", sizeof(short));

    printf("float:  %d 字节\n", sizeof(float));
    printf("double: %d 字节\n", sizeof(double));

}

练习提示

  • 记住整数除法和浮点除法的区别，需要小数结果时确保至少一个操作数是浮点型
  • 字符和整数可以互换使用，理解ASCII码是关键
  • 使用变量前一定记得初始化，避免使用垃圾值
  • 常量名用大写字母，变量名用小写字母，养成好习惯

本章小结

重点回顾

  • 变量是存储数据的容器，使用前必须先声明
  • C语言基本数据类型：int（整数）、float/double（小数）、char（字符）
  • 类型转换分为自动转换（小→大）和强制转换（显式指定）
  • 常量是不可修改的值，可用 const 或 #define 定义
  • 字符本质是整数，通过ASCII编码与数字对应
