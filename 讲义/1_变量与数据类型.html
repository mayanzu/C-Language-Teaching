变量基础

什么是变量？

变量是程序中用来存储数据的容器。你可以把它想象成一个带标签的盒子：

  • 变量名就是盒子上的标签，用来找到这个盒子
  • 变量的值就是盒子里装的东西
  • 变量的类型决定了盒子的大小和能装什么样的东西
生活类比：

想象你要存放不同物品：

  • 整数值（如年龄25）→ 用小盒子（int）
  • 小数值（如身高1.75）→ 用精度盒子（float/double）
  • 单个字符（如字母'A'）→ 用迷你盒子（char）

变量的声明与初始化

使用变量前，必须先声明它——告诉编译器变量的类型和名字。

变量声明与初始化
#include <stdio.h>

int main() {
    // 声明变量（只创建盒子，不放入内容）
    int age;

    // 声明并初始化（创建盒子并放入内容）
    int score = 95;

    // 同时声明多个同类型变量
    int a = 1, b = 2, c = 3;

    // 先声明，后赋值
    float height;
    height = 1.75;

    printf("年龄: %d\n", age);    // 注意：未初始化的变量值不确定！
    printf("成绩: %d\n", score);
    printf("身高: %.2f\n", height);

    return 0;
}
重要警告：

未初始化的局部变量包含垃圾值（不确定的随机值），使用前一定要初始化！

规则 说明 正确示例 错误示例

不能数字开头 第一个字符必须是字母或下划线 _count、num2 2num
不能是关键字 如int、float、return等 myInt int

  • 变量名要有意义，见名知意（如 student_age 而非 a）

整型（Integer）

基本整型 int

int 是C语言中最常用的整数类型，用于存储没有小数部分的数值。

int 类型示例
#include <stdio.h>

int main() {
    int age = 20;          // 年龄
    int temperature = -5;   // 温度（可以是负数）
    int count = 0;           // 计数器

    printf("年龄: %d\n", age);
    printf("温度: %d\n", temperature);
    printf("计数: %d\n", count);

    return 0;
}

整型的分类

C语言提供了多种整型，它们的主要区别是占用的内存大小和能表示的范围：

类型 占用字节 范围 用途
short 2字节 -32768 ~ 32767 节省空间的小整数
int 4字节 -2147483648 ~ 2147483647 一般整数（最常用）
long 4/8字节 更大范围 大整数
long long 8字节 极大范围 非常大的整数
unsigned int 4字节 0 ~ 4294967295 非负整数
新手注意：
  • 日常编程中使用 int 就足够了
  • unsigned 表示无符号，只能存非负数
  • 可以用 sizeof 运算符查看类型占用的字节数
使用sizeof查看类型大小

整数的表示方式

C语言中整数可以用不同的进制表示：

进制 前缀 示例 十进制值
十进制 无 42 42
八进制 0 052 42
十六进制 0x或0X 0x2A 42
不同进制表示
#include <stdio.h>

int main() {
    int dec = 42;      // 十进制
    int oct = 052;      // 八进制（前缀0）
    int hex = 0x2A;     // 十六进制（前缀0x）

    printf("十进制: %d\n", dec);  // 42
    printf("八进制: %d\n", oct);  // 42
    printf("十六进制: %d\n", hex); // 42

    printf("八进制显示: %o\n", dec);   // 52
    printf("十六进制显示: %x\n", dec);  // 2a

    return 0;
}
整数溢出：

当整数超过其类型能表示的最大值时，会发生溢出，结果不可预测：

  • int 最大值约 21亿（2147483647），超过后会"绕回"到负数
  • 就像汽车里程表：到99999后变成00000
  • 避免方法：用 long long 存放大数，或提前检查
整数溢出示例
#include <stdio.h>
#include <limits.h>  // 包含INT_MAX等常量

int main() {
    int max = INT_MAX;  // 2147483647
    printf("int最大值: %d\n", max);
    printf("最大值+1: %d\n", max + 1);  // 溢出！变成负数

    // 用long long安全地存放大数
    long long big = (long long)max + 1;
    printf("安全计算: %lld\n", big);  // 正确: 2147483648

    return 0;
}

浮点型（Float / Double）

什么是浮点型？

浮点型用于存储带小数部分的数值。C语言提供了两种浮点类型：

类型 占用字节 有效数字 范围 用途
float 4字节 6-7位 ±3.4×10³⁸ 单精度，一般小数
double 8字节 15-16位 ±1.7×10³⁰⁸ 双精度，更精确的小数

浮点型示例

浮点型使用示例
#include <stdio.h>

int main() {
    float height = 1.75f;       // 注意：float类型加f后缀
    double pi = 3.141592653589; // double类型更精确
    float temperature = 36.5f;

    printf("身高: %.2f 米\n", height);      // 保留2位小数
    printf("圆周率: %.10f\n", pi);         // 保留10位小数
    printf("体温: %.1f 度\n", temperature); // 保留1位小数

    // 科学计数法
    double distance = 1.5e8;  // 1.5 × 10⁸ = 150000000
    printf("距离: %e\n", distance);  // 科学计数法显示

    return 0;
}
浮点数精度问题：
  • 浮点数在计算机中是近似存储的，不是精确值
  • 0.1 + 0.2 的结果可能不是精确的 0.3
  • 需要高精度时使用 double 而非 float
  • 不要用 == 直接比较两个浮点数是否相等

字符型（char）

什么是字符型？

char 类型用于存储单个字符。在C语言中，字符本质上是一个小整数，因为计算机用数字来表示字符（ASCII编码）。

字符型基本使用
#include <stdio.h>

int main() {
    char grade = 'A';       // 字符用单引号包围
    char digit = '5';       // 字符'5'和数字5不同！
    char symbol = '+';      // 符号也是字符

    printf("等级: %c\n", grade);    // 以字符形式输出
    printf("ASCII值: %d\n", grade);  // 以整数形式输出ASCII码

    return 0;
}

ASCII码表（常用部分）

字符 ASCII值 字符 ASCII值
'0' - '9' 48 - 57 'A' - 'Z' 65 - 90
'a' - 'z' 97 - 122 ' '（空格） 32
'\n'（换行） 10 '\0'（空字符） 0
字符运算：

因为字符本质是整数，所以可以对字符做算术运算：

  • 大写转小写：小写 = 大写 + 32（如 'A' + 32 = 'a'）
  • 字符转数字：数字 = 字符 - '0'（如 '5' - '0' = 5）

转义字符

常用转义字符

有些字符无法直接用键盘输入（如换行、制表符），或者与语法冲突（如引号）。C语言用反斜杠 \ 开头的转义字符来表示这些特殊字符。

转义字符 含义 ASCII值 示例
\n 换行 10 printf("Hello\nWorld");
\t 水平制表符（Tab） 9 printf("A\tB");
\\ 反斜杠本身 92 printf("C:\\Users");
\' 单引号 39 char ch = '\'';
\" 双引号 34 printf("他说\"你好\"");
\0 空字符（字符串结束符） 0 char s[] = "Hi"; 末尾自动加
\r 回车 13 Windows换行：\r\n
\b 退格 8 删除前一个字符
转义字符使用示例
#include <stdio.h>

int main() {
    // 换行与制表
    printf("姓名\t年龄\t成绩\n");
    printf("张三\t20\t89.5\n");

    // 输出引号和反斜杠
    printf("文件路径: C:\\Users\\doc\\file.txt\n");
    printf("他说: \"Hello!\"\n");

    // 单引号字符
    char quote = '\'';
    printf("单引号字符: %c\n", quote);

    return 0;
}
常见错误：
  • 忘记转义反斜杠："C:\new" 实际是 "C:" + 换行 + "ew"！正确写法："C:\\new"
  • 在字符常量中忘记转义引号：char ch = '''; 错误！应为 char ch = '\'';

类型转换

自动类型转换

当不同类型的数据参与运算时，C语言会自动进行类型转换，遵循"小类型→大类型"的原则：

自动类型转换
#include <stdio.h>

int main() {
    int a = 10;
    float b = 3.0f;

    // int和float运算，int自动转为float
    float result = a / b;  // 10.0 / 3.0 = 3.333...
    printf("10 / 3.0 = %.2f\n", result);

    // 但两个int相除，结果还是int
    int int_result = a / 3;  // 10 / 3 = 3（小数部分被丢弃）
    printf("10 / 3 = %d\n", int_result);

    // char参与运算时，自动转为int
    char ch = 'A';
    int ch_value = ch + 1;  // 65 + 1 = 66，即'B'
    printf("'A' + 1 = %c\n", ch_value);

    return 0;
}

强制类型转换

当你需要明确地将一种类型转换为另一种类型时，使用强制类型转换（类型转换运算符）：

强制类型转换
#include <stdio.h>

int main() {
    int a = 10, b = 3;

    // 不转换：整数除法
    printf("整数除法: %d\n", a / b);  // 输出: 3

    // 强制转换：浮点除法
    printf("浮点除法: %.2f\n", (float)a / b);  // 输出: 3.33

    // 也可以转换另一个操作数
    printf("浮点除法: %.2f\n", a / (float)b);  // 输出: 3.33

    // double转int：小数部分被截断
    double pi = 3.14159;
    int int_pi = (int)pi;
    printf("pi = %f, (int)pi = %d\n", pi, int_pi);  // 3.141590, 3

    return 0;
}
类型转换规则：
  • 转换方向：char → short → int → long → float → double
  • 从低精度到高精度：安全，不会丢失数据
  • 从高精度到低精度：可能丢失数据（如 double → int 会截断小数）

常量

什么是常量？

常量是程序运行过程中值不能改变的数据。与变量不同，常量一旦定义就不能修改。

常量的定义方式

1. const 关键字
  • 声明时加 const 修饰
  • 有明确的数据类型
  • 可以被调试器检查
  • 推荐使用这种方式
2. 宏定义 #define
  • 编译前文本替换
  • 没有类型检查
  • 不占内存空间
  • 常用于全局常量
常量定义示例
#include <stdio.h>

// 方式1：宏定义（习惯用大写字母）
#define PI 3.14159
#define MAX_SIZE 100

int main() {
    // 方式2：const关键字
    const double E = 2.71828;
    const int DAYS_PER_WEEK = 7;

    // 使用常量
    double area = PI * 5.0 * 5.0;
    printf("圆面积: %.2f\n", area);
    printf("自然常数: %f\n", E);
    printf("每周天数: %d\n", DAYS_PER_WEEK);

    // DAYS_PER_WEEK = 8;  // 错误！常量不能修改

    return 0;
}

布尔类型

什么是布尔类型？

在C语言中，布尔类型用于表示逻辑上的"真"和"假"。C89标准中没有专门的布尔类型，而是用整数来表示：0表示假，非0表示真。

C99的 _Bool 类型

C99标准引入了 _Bool 关键字，它只能存储0或1：

_Bool 类型使用
#include <stdio.h>

int main() {
    _Bool is_student = 1;   // 真（是学生）
    _Bool is_old = 0;       // 假（不老）

    // 任何非0值赋给_Bool都会变成1
    _Bool result = 42;      // result 的值是 1

    printf("是学生: %d\n", is_student);
    printf("是否年长: %d\n", is_old);
    printf("结果: %d\n", result);

    return 0;
}

stdbool.h 头文件

使用 stdbool.h 可以让布尔类型更易读：

使用 stdbool.h
#include <stdio.h>
#include <stdbool.h>   // 引入 bool、true、false

int main() {
    bool is_pass = true;    // 比用1更易读
    bool is_fail = false;   // 比用0更易读

    printf("通过: %d\n", is_pass);   // 输出: 通过: 1
    printf("挂科: %d\n", is_fail);   // 输出: 挂科: 0

    return 0;
}
要点总结：
  • 传统C语言用整数表示真假：0为假，非0为真
  • C99的 _Bool 只能存0或1
  • stdbool.h 提供了更易读的 bool、true、false
  • 在后续学习条件判断时，布尔类型会大量使用

数据类型选择指南

面对不同需求，应该如何选择合适的数据类型？

需求场景 推荐类型 原因
一般整数（年龄、数量等）int最常用，效率最高
大整数（人口、金额等）long long范围足够大
非负整数（学号、编号）unsigned int范围翻倍
一般小数double精度更高，默认推荐
节省内存的小数float4字节，适合大量数据
单个字符char1字节，专为字符设计
真/假判断int 或 boolC89用int，C99可用bool
常见误区：
  • 不要用 float 来"节省空间"——现代计算机上 double 的运算速度往往更快
  • 不要用 short 来"节省空间"——32位/64位CPU上 int 运算通常更快
  • 表示金额时不要用浮点型——浮点数有精度问题，实际项目中用整数（分）来表示

limits.h 与 float.h

C标准库提供了两个头文件，包含各类型的取值范围和精度信息：

使用 limits.h 查看整型范围
#include <stdio.h>
#include <limits.h>

int main() {
    printf("int 最小值: %d\n", INT_MIN);
    printf("int 最大值: %d\n", INT_MAX);
    printf("unsigned int 最大值: %u\n", UINT_MAX);
    printf("char 位数: %d\n", CHAR_BIT);

    return 0;
}
实用建议：

当你的程序需要处理很大或很小的数值时，先查阅这些头文件中的常量，确保所选类型的范围足够。

  • 学生成绩：int score; — 0-100是整数范围，int足够
  • 银行余额：long long balance; — 以分为单位的整数，避免浮点精度问题
  • 城市人口：unsigned int population; — 人口不可能为负，unsigned int范围0~42亿
  • 是否登录：int is_logged_in; — 0/1表示真/假

本章小结

重点回顾

  • 变量是存储数据的容器，使用前必须先声明
  • C语言基本数据类型：int（整数）、float/double（小数）、char（字符）
  • 类型转换分为自动转换（小→大）和强制转换（显式指定）
  • 常量是不可修改的值，可用 const 或 #define 定义
  • 字符本质是整数，通过ASCII编码与数字对应
  • 转义字符用 \ 开头表示特殊字符：\n（换行）、\t（制表）、\\（反斜杠）等
  • 整数溢出：超过类型范围时结果不可预测，用 limits.h 中的常量检查
  • C99引入 _Bool 类型和 stdbool.h，提供更清晰的布尔值表示
  • 选择数据类型时应根据实际需求：一般用int和double，注意溢出和精度问题
  • 良好的变量命名应语义清晰，推荐snake_case风格
