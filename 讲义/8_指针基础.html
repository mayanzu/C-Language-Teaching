指针基础

理解内存地址的概念，掌握指针的基本操作

简介

指针是C语言的精髓，也是它区别于其他高级语言的重要特征。很多人觉得指针难，那是因为没有理解它的本质——指针就是地址，存储另一个变量在内存中的位置。

理解指针后，你就能：直接操作内存、按引用传递数据、高效处理数组和字符串……甚至能理解操作系统如何管理内存。反之，不懂指针，C语言只学了一半。

新手注意：
  • 指针存储的是地址，不是数据本身
  • & 取地址，* 解引用（取地址处的值）
  • 指针需要初始化，否则可能指向未知内存（野指针）
  • 指针的大小取决于系统：32位系统4字节，64位系统8字节
生活类比：房间与房卡
  • 内存 = 大楼里的所有房间
  • 变量 = 房间里的物品
  • 指针 = 房卡（存储房间号）
  • 解引用 = 用房卡打开门，拿到物品
生活类比：快递柜
  • 内存地址 = 快递柜格口号（如"3-15"）
  • 变量值 = 格子里放的包裹
  • 指针 = 记录格口号的小纸条
  • 解引用 = 打开纸条上写的格子，取出包裹

指针的概念

内存地址

计算机内存由大量字节（byte）组成，每个字节都有一个唯一的编号，这就是内存地址。就像酒店的房间号，每个房间（字节）都有独一无二的房号（地址）。

重要理解：

当你声明一个变量时，计算机会在内存中分配一块空间给这个变量，并记住这块空间的地址。指针就是用来存储这个地址的变量。

内存与地址示意图
// 假设 int a = 100;
// 变量a在内存中占用4个字节（假设从地址0x1000开始）

// 地址示意图：
// 0x1000: [第1个字节]
// 0x1001: [第2个字节]
// 0x1002: [第3个字节]
// 0x1003: [第4个字节]

// &a 就是 0x1000（变量a的起始地址）

指针的本质

指针是一个变量，它存储的是另一个变量的内存地址。指针本身也有地址，就像房卡也有自己的放置位置。

概念 说明 类比
普通变量 存储数据值 房间里的物品
指针变量 存储地址值 记录房间号的小纸条
解引用 通过指针访问指向的数据 按房号找房间、取物品

指针声明与初始化

声明语法

指针的声明语法是在变量名前加一个星号 *。指针的类型必须与它指向的变量的类型匹配。

指针类型 指向的类型 示例
int *p 指向 int 存储整型变量的地址
char *p 指向 char 存储字符变量的地址
double *p 指向 double 存储浮点变量的地址
int *p1, *p2 多个指针 每个指针前都要加 *
注意：

int *p1, p2; 中 p1 是指针，但 p2 是普通整型变量！每个指针声明都要加 *。建议分开声明：int *p1, *p2;

初始化

指针必须初始化才能使用，否则它可能指向任意内存地址（野指针），导致程序崩溃或数据损坏。

指针的初始化方式
#include <stdio.h>

int main() {
    int a = 10;
    int *p;          // 声明指针（未初始化，危险！）
    p = &a;             // 初始化：让p指向a的地址

    // 也可以声明时直接初始化
    int b = 20;
    int *ptr = &b;  // 推荐写法：声明和初始化一起

    // 空指针：暂时不指向任何地址
    int *empty = NULL;  // NULL表示"空"，不指向任何有效地址

    return 0;
}
建议：
  • 声明指针时尽量同时初始化
  • 如果暂时不知道指向谁，用 NULL 初始化
  • 使用指针前先检查是否为 NULL

练习1： 写出以下代码的输出结果，并解释原因：

分析代码
#include <stdio.h>

int main() {
    int a = 5, b = 10;
    int *p1 = &a;
    int *p2 = &b;

    printf("*p1 + *p2 = %d\n", *p1 + *p2);
    printf("*p1 * *p2 = %d\n", *p1 * *p2);
    return 0;
}
参考答案

输出：

*p1 + *p2 = 15（5 + 10）

*p1 * *p2 = 50（5 × 10）

原因：*p1 和 *p2 分别取得两个指针指向的值，即a和b的值。

取地址运算符 &

&运算符

& 是一元运算符，放在变量前面，返回该变量的内存地址。

&运算符使用
#include <stdio.h>

int main() {
    int a = 100;
    int *p = &a;  // &a 取得变量a的地址，赋值给指针p

    printf("变量a的值: %d\n", a);
    printf("变量a的地址: %p\n", &a);
    printf("指针p的值(存的地址): %p\n", p);
    printf("指针p的地址: %p\n", &p);

    return 0;
}
输出示例：

变量a的值: 100

变量a的地址: 0x7ffd5a3b4c44

指针p的值(存的地址): 0x7ffd5a3b4c44

指针p的地址: 0x7ffd5a3b4c48（指针本身也有地址）

边学边练 - 取地址运算符 &

练习2： 编写一个函数，使用指针参数返回数组中的最大值和最小值。

参考答案
查找最大最小值
#include <stdio.h>

void findMinMax(int arr[], int n, int *min, int *max) {
    *min = arr[0];
    *max = arr[0];
    for (int i = 1; i < n; i++) {
        if (arr[i] < *min) *min = arr[i];
        if (arr[i] > *max) *max = arr[i];
    }
}

int main() {
    int arr[] = {3, 7, 2, 9, 1};
    int min, max;
    findMinMax(arr, 5, &min, &max);
    printf("最小值: %d, 最大值: %d\n", min, max);
    return 0;
}

解引用运算符 *

*运算符（解引用）

* 是解引用运算符，放在指针前面，返回该指针指向的变量的值。通俗地说：*指针就是"指针指向的那个变量"。

理解解引用：

指针里存的是地址，就像房卡上写的房间号。*ptr 就是"去打开房卡上写的那个房间，告诉我里面有什么"。

解引用操作
#include <stdio.h>

int main() {
    int a = 100;
    int *p = &a;

    printf("a的值: %d\n", a);       // 直接访问a：100
    printf("*p的值: %d\n", *p);      // 解引用p：也是100

    // 通过指针修改a的值
    *p = 200;  // 等价于 a = 200;
    printf("修改后*a: %d\n", *p);  // 200
    printf("修改后a: %d\n", a);    // 200

    return 0;
}
重要：

解引用一个未初始化或为 NULL 的指针会导致程序崩溃！就像拿着一个无效的房卡去开门，后果很严重。

&和*的关系

操作 含义 结果类型
&a 取变量a的地址 指针（如 int *）
*p 取指针p指向的变量的值 指针指向的类型（如 int）
*&a 取地址再解引用 = 原值 int（等于a）
&*p 解引用再取地址 = 原地址 int *（等于p）
&和*互为逆运算
#include <stdio.h>

int main() {
    int a = 10;
    int *p = &a;

    printf("*&a = %d\n", *&a);  // 等于a的值：10
    printf("&*p = %p\n", &*p);   // 等于p的值：a的地址
    return 0;
}

指针与数组

数组名就是首地址

在C语言中，数组名就是数组首元素的地址。这是指针与数组紧密联系的基础。

数组名与指针的关系
#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};

    printf("arr = %p\n", arr);        // 数组首地址
    printf("&arr[0] = %p\n", &arr[0]);  // 第一个元素的地址
    // 两者相等！

    // 指针p指向数组首元素
    int *p = arr;  // 等价于 int *p = &arr[0];

    return 0;
}

指针算术运算

指针可以加减整数。加1表示指向下一个同类型元素，减1表示指向上一个元素。

操作 含义 结果
p + n 指向下n个元素 &arr[n]
p - n 指向上n个元素 &arr[-n]
p++ 指向下一个元素 &arr[i+1]
*(p + n) 取下n个元素的值 arr[n]
重要规律：

arr[i] 等价于 *(arr + i)

数组名（首地址）+ 偏移量 = 目标地址

用指针遍历数组
#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *p = arr;
    int n = 5;

    // 方法1：用下标访问
    printf("方法1: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    // 方法2：用指针遍历
    printf("方法2: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", *(p + i));  // 等价于arr[i]
    }
    printf("\n");

    return 0;
}

指针与数组名的区别

特性 数组名 arr 指针 p
类型 常量指针，不能赋值 变量，可以改变指向
值 首元素地址 存储的地址
sizeof 数组总大小 指针变量大小（8字节）
sizeof 的区别
#include <stdio.h>

int main() {
    int arr[5];
    int *p = arr;

    printf("sizeof(arr) = %d\n", sizeof(arr));  // 20 (5个int × 4字节)
    printf("sizeof(p) = %d\n", sizeof(p));        // 8 (指针大小，64位系统)

    return 0;
}

练习3： 用指针实现字符串长度计算（不使用 strlen）。

参考答案
计算字符串长度
#include <stdio.h>

int myStrlen(char *str) {
    int len = 0;
    while (*str != '\0') {
        len++;
        str++;  // 指针后移
    }
    return len;
}

int main() {
    char s[] = "Hello";
    printf("长度: %d\n", myStrlen(s));  // 5
    return 0;
}

指针作为函数参数

值传递 vs 地址传递

在函数章节我们学过，C语言是值传递——形参是实参的副本。但如果我们传地址过去，虽然还是"复制地址"，但复制的是地址本身，通过这个地址可以访问和修改原始数据！

类比：

值传递 = 给复印件（改复印件不影响原件）

地址传递 = 给房卡复印件（虽然卡是复制品，但用它能打开同一个房间、改动里面的东西）

用指针实现数据交换

之前我们学过，swap(a, b) 无法交换两个变量的值，因为是值传递。但用指针就可以：

用指针交换两个变量的值
#include <stdio.h>

// 交换两个整数的值（通过指针）
void swap(int *x, int *y) {
    int temp = *x;   // temp = a的值
    *x = *y;             // a = b的值
    *y = temp;           // b = temp
}

int main() {
    int a = 3, b = 5;
    printf("交换前: a = %d, b = %d\n", a, b);

    swap(&a, &b);  // 传地址过去

    printf("交换后: a = %d, b = %d\n", a, b);
    return 0;
}
输出：

交换前: a = 3, b = 5

交换后: a = 5, b = 3

用指针返回多个值

函数只能有一个返回值，但通过指针参数，可以"返回"多个值。

计算商和余数
#include <stdio.h>

// 计算除法的商和余数
void divide(int a, int b, int *quotient, int *remainder) {
    *quotient = a / b;
    *remainder = a % b;
}

int main() {
    int q, r;
    divide(17, 5, &q, &r);
    printf("17 ÷ 5 = %d ... %d\n", q, r);
    return 0;
}

练习4： 以下代码有什么问题？如何修正？

找错题
#include <stdio.h>

int *getPointer() {
    int num = 42;
    return #
}

int main() {
    int *ptr = getPointer();
    printf("%d\n", *ptr);
    return 0;
}
参考答案

问题：返回了局部变量的地址。函数 getPointer() 结束时，局部变量 num 被销毁，指针指向一个不再有效的内存地址。

修正方法：

  • 使用全局变量或静态变量
  • 让调用者传入指针，函数填充数据
  • 动态分配内存（后续章节）

指针与字符串

字符指针与字符数组

C语言中，字符串可以用字符数组或字符指针来表示，但两者有重要区别：

字符数组 vs 字符指针
#include <stdio.h>

int main() {
    // 方式1：字符数组（可修改）
    char arr[] = "Hello";
    arr[0] = 'h';         // 正确！可以修改
    printf("%s\n", arr);   // hello

    // 方式2：字符指针（指向常量区，不可修改！）
    char *ptr = "Hello";
    // ptr[0] = 'h';  // 错误！字符串常量不可修改
    printf("%s\n", ptr);   // Hello

    return 0;
}
特性 char arr[] = "Hello" char *ptr = "Hello"
存储位置 栈（可修改） 字符串常量区（不可修改）
能否修改内容 可以 不能！修改会导致崩溃
sizeof 数组大小（6字节） 指针大小（8字节）
推荐用途 需要修改字符串时 只读字符串、字符串参数

const指针

三种const指针

const 和指针的组合有三种情况，初学者容易混淆。关键看 const 修饰的是谁：

声明 含义 能改什么 不能改什么
const int *p 指向常量的指针 p的指向（让p指别人） *p的值（不能通过p改数据）
int * const p 指针本身是常量 *p的值（改指向的数据） p的指向（p永远指向同一个地址）
const int * const p 都不可改 都不能改 都不能改
记忆技巧：

看 const 在 * 的左边还是右边：

  • const 在 * 左边 → 修饰指向的数据，数据不可改
  • const 在 * 右边 → 修饰指针本身，指向不可改
const指针示例
#include <stdio.h>

int main() {
    int a = 10, b = 20;

    // 1. 指向常量的指针：不能通过p修改a
    const int *p1 = &a;
    // *p1 = 30;  // 错误！不能通过p1修改
    p1 = &b;      // 正确！可以改变指向

    // 2. 常量指针：p2永远指向a
    int * const p2 = &a;
    *p2 = 30;    // 正确！可以修改指向的数据
    // p2 = &b;  // 错误！不能改变指向

    return 0;
}
实际应用：

函数参数中常用 const int *p 来承诺"我不会修改你传给我的数据"，让调用者放心。例如：void printArray(const int *arr, int n) 表示这个函数只读不写。

常见错误

空指针与野指针

类型 定义 危险性
野指针 未初始化的指针 指向随机地址，极易崩溃
空指针 指向NULL（地址0） 安全，可检查，但解引用会崩溃
野指针与空指针
#include <stdio.h>

int main() {
    int *wild;          // 野指针！没有初始化
    int *empty = NULL;  // 空指针，安全

    // 使用前检查
    if (empty != NULL) {
        printf("%d\n", *empty);
    } else {
        printf("指针为空\n");
    }

    return 0;
}

指针使用注意事项

必须避免的错误：
  • 野指针：指针声明后不初始化就使用
  • 空指针解引用：对NULL进行 * 操作
  • 越界访问：指针超出了数组范围
  • 类型不匹配：用 int * 指向 double
  • 返回局部变量地址：函数结束后局部变量被销毁
错误示例
// 错误1：野指针
int *p;       // 未初始化！
*p = 10;     // 危险！

// 错误2：空指针解引用
int *p = NULL;
*p = 10;     // 崩溃！

// 错误3：返回局部变量地址
int *badFunc() {
    int local = 100;
    return &local;  // 危险！函数结束后local被销毁
}

练习5： 编写一个函数，使用指针反转一个整型数组。

参考答案
反转数组
#include <stdio.h>

void reverse(int *arr, int n) {
    int *start = arr;
    int *end = arr + n - 1;
    while (start < end) {
        int temp = *start;
        *start = *end;
        *end = temp;
        start++;
        end--;
    }
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    reverse(arr, 5);
    for (int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }  // 输出: 5 4 3 2 1
    return 0;
}
