函数调用与定义练习

掌握 C 语言函数的定义、声明和调用方法

函数基础

函数是C语言程序的基本组成单元，用于封装可重复使用的代码块。每个函数都有特定的功能，通过函数名、参数和返回值进行交互。

新手注意：
  • 函数必须先声明后使用
  • main函数是程序的入口点
  • 函数可以调用其他函数

函数定义

函数的组成结构

什么是函数？

函数是C语言中用于封装特定功能代码的基本单元。可以把函数理解为一个"黑盒子"：给它输入（参数），它经过处理后给出输出（返回值）。使用函数可以让代码更加模块化、易于维护和复用。

为什么要使用函数？

  • 代码复用：相同的功能不需要重复编写，一次定义多次使用
  • 模块化：将复杂问题分解成多个小问题，逐个解决
  • 易于维护：修改功能时只需修改对应的函数，不影响其他代码
  • 提高可读性：良好命名的函数让代码意图更清晰

函数的四个组成部分

每个函数都由四个基本部分组成：

  • 返回值类型：函数返回的数据类型（int, double, void等）
  • 函数名：遵循标识符命名规则，建议使用动词或动词短语
  • 参数列表：函数接收的输入数据（可以为空）
  • 函数体：实现功能的代码块，用大括号 {} 包围
函数基本格式
// 函数定义的基本格式
返回值类型 函数名(参数类型1 参数名1, 参数类型2 参数名2) {
    // 函数体：实现功能的代码
    return 返回值;  // void类型不需要return
}

简单示例：求两数之和

有参数有返回值的函数
#include <stdio.h>

// 函数定义：计算两数之和
int add(int a, int b) {
    return a + b;  // 返回结果
}

int main() {
    int sum = add(5, 3);  // 调用函数
    printf("5 + 3 = %d\n", sum);
    return 0;
}
⚠️ 函数定义的注意事项：
  • 函数名应该清晰表达功能（如calculateSum而不是func1）
  • 形参和实参的数量、类型必须匹配
  • return语句会立即结束函数并返回值
  • void函数不能返回值，其他类型函数必须return对应类型的值

函数声明

函数原型声明

什么是函数声明？

函数声明（也叫函数原型）是告诉编译器函数存在的一种方式。它只描述函数的接口（返回值类型、函数名、参数类型），不包含函数的具体实现。

为什么需要函数声明？

  • 编译顺序：C语言从上到下编译，如果函数定义在调用之后，需要先声明
  • 代码组织：可以把函数定义放在文件末尾，保持main函数在前面，便于阅读
  • 多文件开发：在头文件中声明函数，在源文件中定义，实现模块化编程

函数声明的格式

函数声明的基本格式
// 格式：返回值类型 函数名(参数类型列表);
int add(int, int);      // 可以省略参数名
int add(int a, int b);  // 也可以带参数名
void greet(char[]);        // 数组参数

简单示例

函数声明示例
#include <stdio.h>

// 函数声明
int max(int, int);

int main() {
    int a = 10, b = 20;
    printf("较大值: %d\n", max(a, b));
    return 0;
}

// 函数定义在后面
int max(int x, int y) {
    return x > y ? x : y;
}

函数调用

函数调用的方式

如何调用函数？

函数调用就是使用已经定义好的函数。调用时需要提供函数名和必要的参数（实参），函数执行后可能会返回一个值。

函数调用的几种方式

  • 赋值方式：将函数返回值赋给变量
  • 表达式方式：直接在表达式中调用函数
  • 语句方式：单独作为一条语句（常用于void函数）
函数调用示例
#include <stdio.h>

int square(int n) {
    return n * n;
}

int main() {
    // 方式1：赋值给变量
    int result = square(5);
    printf("5的平方: %d\n", result);
    
    // 方式2：直接用在表达式中
    printf("3的平方: %d\n", square(3));
    
    // 方式3：作为参数传递
    printf("结果: %d\n", square(2) + square(3));
    
    return 0;
}

int createAccount(char accounts[][20], double balances[], int *count) {

参数传递

值传递与地址传递

什么是参数传递？

参数传递是指将实参的值传递给函数形参的过程。C语言有两种主要的参数传递方式：值传递和地址传递。

两种传递方式的区别

  • 值传递：传递变量的副本，函数内修改不影响原变量
  • 地址传递：传递变量的地址（指针），函数内可以修改原变量
值传递 vs 地址传递
#include <stdio.h>

// 值传递：不能修改原变量
void modifyValue(int x) {
    x = x * 2;
    printf("函数内x: %d\n", x);
}

// 地址传递：可以修改原变量
void modifyPtr(int *x) {
    *x = *x * 2;
}

int main() {
    int a = 10, b = 10;
    
    modifyValue(a);     // a不变，仍为10
    modifyPtr(&b);      // b改变，变为20
    
    printf("a=%d, b=%d\n", a, b);
    return 0;
}

数组作为函数参数

数组参数的特点

数组作为函数参数时，传递的是数组首地址（指针），因此函数内可以修改原数组的元素。同时需要传递数组大小，因为函数无法自动获知数组长度。

数组参数示例
#include <stdio.h>

// 计算数组元素之和
int arraySum(int arr[], int size) {
    int sum = 0;
    for(int i = 0; i < size; i++) {
        sum += arr[i];
    }
    return sum;
}

int main() {
    int nums[] = {1, 2, 3, 4, 5};
    printf("数组和: %d\n", arraySum(nums, 5));
    return 0;
}

综合参数传递应用
#include <stdio.h>
#include <string.h>

// 学生结构体
typedef struct {
    char name[50];
    int id;
    int scores[3];  // 三科成绩
} Student;

// 函数声明
void inputStudent(Student *s);
void printStudent(const Student *s);
double calculateAverage(const int scores[], int count);
char getGrade(double average);
void sortStudentsByAverage(Student students[], int count);

int main() {
    Student students[3];
    
    printf("=== 学生信息录入 ===\n");
    for(int i = 0; i < 3; i++) {
        printf("\n录入第%d个学生信息:\n", i+1);
        inputStudent(&students[i]);  // 结构体指针传递
    }
    
    printf("\n=== 学生信息显示 ===\n");
    for(int i = 0; i < 3; i++) {
        printStudent(&students[i]);  // 结构体指针传递
    }
    
    sortStudentsByAverage(students, 3);  // 数组传递
    
    printf("\n=== 按平均分排序后 ===\n");
    for(int i = 0; i < 3; i++) {
        printStudent(&students[i]);
    }
    
    return 0;
}

void inputStudent(Student *s) {
    printf("姓名: ");
    scanf("%s", s->name);
    printf("学号: ");
    scanf("%d", &s->id);
    printf("三科成绩 (语文 数学 英语): ");
    for(int i = 0; i < 3; i++) {
        scanf("%d", &s->scores[i]);
    }
}

void printStudent(const Student *s) {
    double avg = calculateAverage(s->scores, 3);  // 数组传递
    char grade = getGrade(avg);
    
    printf("姓名: %s, 学号: %d\n", s->name, s->id);
    printf("成绩: %d %d %d, 平均分: %.1f, 等级: %c\n", 
           s->scores[0], s->scores[1], s->scores[2], avg, grade);
}

double calculateAverage(const int scores[], int count) {
    int sum = 0;
    for(int i = 0; i < count; i++) {
        sum += scores[i];
    }
    return (double)sum / count;
}

char getGrade(double average) {
    if(average >= 90) return 'A';
    else if(average >= 80) return 'B';
    else if(average >= 70) return 'C';
    else if(average >= 60) return 'D';
    else return 'F';
}

void sortStudentsByAverage(Student students[], int count) {
    for(int i = 0; i < count-1; i++) {
        for(int j = 0; j < count-i-1; j++) {
            double avg1 = calculateAverage(students[j].scores, 3);
            double avg2 = calculateAverage(students[j+1].scores, 3);
            
            if(avg1 < avg2) {
                Student temp = students[j];
                students[j] = students[j+1];
                students[j+1] = temp;
            }
        }
    }
}

递归函数

递归的基本概念

什么是递归？

递归是指函数调用自己的编程技巧。递归函数通过将大问题分解成相同类型的小问题来解决，直到问题简单到可以直接求解为止。

递归的两个要素

  • 递归终止条件：必须有明确的结束条件，否则会无限递归
  • 递归调用：函数调用自己，且每次调用都向终止条件靠近
简单递归示例
#include <stdio.h>

// 计算阶乘：n! = n × (n-1)!
int factorial(int n) {
    if (n <= 1) return 1;  // 终止条件
    return n * factorial(n - 1);  // 递归调用
}

int main() {
    printf("5! = %d\n", factorial(5));  // 120
    return 0;
}
